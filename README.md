# Django Tutorial 01

Proyecto del Tutorial 01 de Django con aplicación "pages".

## Instalación
1. `pip install -r requirements.txt`
2. `python manage.py migrate`
3. `python manage.py runserver`

- Home: /
- About: /about/ 
- Contact: /contact/
- Products: /products/
